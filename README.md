# 🐵 Monkey Hot Spring — Student Point Tracker

A cute watercolor-themed point tracking system for classrooms. Teachers manage students and award points by clicking on animated snow monkeys relaxing in a hot spring. Students can view their points, see a leaderboard, and play a daily Wordle challenge for bonus points.

## Setup

1. Clone this repo
2. Run `npm install`
3. Run `npm run dev` to test locally
4. Push to GitHub — Vercel will auto-deploy

## Tech Stack

- React + Vite
- Firebase Firestore (database)
- Deployed on Vercel

## Default Login

- **Teacher:** username `teacher` / password `1234`
- **Students:** created by the teacher
